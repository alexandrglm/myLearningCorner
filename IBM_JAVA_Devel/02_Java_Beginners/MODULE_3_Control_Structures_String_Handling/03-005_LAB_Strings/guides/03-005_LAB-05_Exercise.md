


# **Practice Exercise**

- Create two String literals **"Maple Tree"** and compare them using `==`.
- Create a new String object **"Maple Tree"**. Compare with the literal using `==`, then using `.equals`.
- Use **substring** to separate **"Maple"** and **"Tree"**.
- Use **concat** to join them back into **"Maple Tree"**.
- Use **toLowerCase** to convert all characters to lowercase.
- Use **toUpperCase** to convert all characters to uppercase.
