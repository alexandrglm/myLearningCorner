# 03-008 LAB 03 - Conclusion

## Key Learnings

In this lab, you learned how to:

- Add methods to a class which accept parameters and return values
- Create `static` methods that don't require object instantiation
- Use `private` access modifiers to restrict method visibility
- Accept user input and convert between different data types (`String` to `int`, `float`)
- Invoke methods with parameters and utilise return values
- Build an interactive menu-driven programme using loops and conditional statements
