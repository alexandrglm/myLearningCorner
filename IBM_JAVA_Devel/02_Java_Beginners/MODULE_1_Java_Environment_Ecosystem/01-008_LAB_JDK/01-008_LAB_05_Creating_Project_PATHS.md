# Lab 05 - Practice Exercise

## Objective

Create a Java project directory with proper structure and compile/run a program using classpath configuration.

## Tasks

### 1. Create Project Directory Structure

Create a Java project directory named `practice_proj` with `src` and `classes` subdirectories in it.

### 2. Create Java Source File

Create a file in the `src` folder that will print the message:

```
Now I can write a Java program on my own!
```

### 3. Compile with Output Directory

Compile the Java file in such a way that the class file is output in the `classes` directory.

### 4. Set Classpath and Run

Set the classpath and run the file.
