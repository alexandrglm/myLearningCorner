# MODULE 3:     Collections

This module explores the Collections Framework. You'll focus on the Java Collections Framework and the use of lists, sets, queues, and maps for dynamic data management. You’ll develop the knowledge and skills required to effectively use collections in real-world situations. 

## Learning Objectives

-   Identify the components of the Java Collections Framework and their purpose
-   Differentiate between various collection types, such as lists, sets, queues, and maps
-   Apply collection classes to efficiently store, retrieve, and manage data in Java programs
-   Analyse real-world scenarios to determine the appropriate collection type for a given problem
