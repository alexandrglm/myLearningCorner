# Object-Oriented Programming in Java Course

Hello, and welcome to this Object-Oriented Programming in Java course.

This Object-Oriented Programming (OOP) course empowers beginner and experienced software developers with the skills to build robust and scalable Java applications. It provides an in-depth exploration of OOP principles, including encapsulation, inheritance, polymorphism, and abstraction, enabling you to create modular and maintainable code.

You will gain hands-on experience designing classes, implementing constructors, and mastering advanced OOP concepts such as interfaces and abstract classes. You'll learn how to use the Java Collections Framework lists, sets, queues, and maps for efficient data management in complex applications.

Additionally, you will learn essential file and directory management techniques, work with dates and times, and learn how to integrate external services to enhance application functionality. 

This course emphasises building code that implements best practices in software design, such as modularity, code readability, and exception handling, as well as code-building practices that prepare you for real-world software development challenges. As a new developer, use the coding cheat sheets, designed especially for beginners, to gain a detailed understanding of the code you explored in each video.  

In addition to guided hands-on projects, this course offers optional, self-guided "challenge" lab exercises that align with what you learn in each module. Use the provided instructions to develop code using what you know to create the described "challenge" apps. After completing each self-guided lab exercise, validate your code by reviewing the provided solutions. 

Also, take advantage of the Case Study labs, which provide intermediate-level instructions for building a real-world human resources application for managing employees' time off from work. This project is ideal for sharing with prospective employers.

This course is part of the IBM Java Developer Professional Certificate, a structured learning path designed to provide practical programming experience. It is an ideal starting point for aspiring Java developers.

## Prerequisites and Hardware Requirements

This course is recommended for budding and experienced software developers who want to understand object-oriented programming (OOP) in Java. You will need a laptop or desktop with internet access to access the course materials.

## Course Objectives

After completing this course, you will be able to:

- Apply Object-Oriented Programming (OOP) concepts
- Define and implement classes and objects, demonstrating encapsulation and abstraction
- Use advanced OOP concepts such as inheritance, polymorphism, interfaces, and method overloading
- Employ the Java collections framework to manage data using lists, sets, queues, and maps
- Read and write to files using Java streams
- Handle directories effectively
- Work with Java date and time classes for effective date manipulation and formatting
- Integrate external services and call external APIs
- Demonstrate your knowledge with the completion of a comprehensive Java programming project

## Course Outline

This course consists of six modules.

### Module 1: Object-Oriented Programming Basics in Java

In this module, you will learn the core principles of Object-Oriented Programming (OOP) in Java, including classes, objects, encapsulation, and constructors. You will also explore advanced concepts such as inheritance, polymorphism, interfaces, and abstract classes. Through hands-on labs, you will create classes, build constructors, implement polymorphism, and design interfaces and abstract classes. By the end of this module, you will be able to apply OOP principles effectively in Java programmes, ensuring robust and modular code design.

### Module 2: Advanced Object-Oriented Programming Concepts in Java

This module delves into advanced OOP concepts such as inheritance, polymorphism, interfaces, and inner classes in Java. You will learn to compare and apply these concepts in real-world scenarios, gaining a deeper understanding of their uses and best practices. Through practical labs, you will implement polymorphism and design interfaces and abstract classes, enhancing your ability to create flexible and maintainable Java applications.

### Module 3: Working with Collections in Java

In this module, you will explore the Java Collections Framework, focusing on using lists, sets, queues, and maps for dynamic data management. You will learn to identify and differentiate between various collection types and apply them to efficiently store, retrieve, and manage data. Through hands-on labs and real-world scenarios, you will develop practical skills in choosing the appropriate collection type for different programming challenges.

### Module 4: File and Directory Management in Java

This module focuses on file handling in Java, including creating, reading, writing files, and managing directories. You will gain hands-on experience using different types of file streams for input and output operations and learn to implement piped streams. By the end of this module, you will be equipped with the essential file management skills required for building robust Java applications.

### Module 5: Date and Time Handling in Java

In this module, you will explore Java's built-in date and time classes to format dates, display dates, and manage time zones. You will learn to calculate differences between dates and handle time zones effectively. Mastering these skills lets you implement date and time functionalities in real-world applications, ensuring accurate time-based calculations and data management.

### Module 6: Final Project and Course Summary

In this final module, you will apply all the Java skills you acquired throughout the course to develop a personal mood-tracking application. This project involves securely creating, editing, and managing mood entries, using Java Collections for data storage, implementing file handling for data persistence, and managing dates for accurate diary entries. This project also includes optional features such as search functionality, mood statistics, and password protection, enabling you to build a comprehensive and user-friendly application.
