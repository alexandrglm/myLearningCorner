# Métodos abreviados Office 2016

## 1. Word 2016

---

# 1.1 Acceso general, archivo, edición y formato

| Acción | Atajo |
|----------|----------|
| Activar acceso por teclado a la cinta | `Alt` |
| Deshacer una acción | `Ctrl` + `Z` |
| Rehacer o repetir una acción | `Ctrl` + `Y` |
| Aplicar negrita | `Ctrl` + `N` |
| Aplicar cursiva | `Ctrl` + `K` |
| Aplicar subrayado | `Ctrl` + `S` |
| Reducir tamaño de fuente | `Ctrl` + `<` |
| Aumentar tamaño de fuente | `Ctrl` + `Mayús` + `>` |
| Eliminar formato del texto o párrafo | `Ctrl` + `Barra espaciadora` |
| Copiar | `Ctrl` + `C` |
| Cortar | `Ctrl` + `X` |
| Pegar | `Ctrl` + `V` |
| Pegado especial | `Ctrl` + `Alt` + `G` |
| Crear nuevo documento | `Ctrl` + `U` |
| Abrir documento | `Ctrl` + `A` |
| Cerrar documento | `Ctrl` + `R` |
| Cerrar documento activo | `Ctrl` + `F4` |
| Guardar documento | `Ctrl` + `G` |
| Imprimir documento | `Ctrl` + `P` |
| Salir de Word | `Alt` + `F4` |
| Ayuda | `F1` |

> ⚠️ El PDF recoge dos formas de cerrar documentos:
>
> - `Ctrl` + `R`
> - `Ctrl` + `F4`
>
> Ambas aparecen en la documentación.

# 	1.2 Eliminación, selección y edición

|----------|----------|
| Desactivar Copiar formato | `Esc` |
| Borrar filas/columnas/celdas seleccionadas en tablas | `Retroceso` |
| Borrar contenido de celdas | `Supr` |
| Borrar salto de sección | `Supr` |
| Eliminar nota al pie o final | `Supr` o `Retroceso` |

# 	1.3 Desplazamiento por el documento

## Movimiento básico

| Acción | Atajo |
|----------|----------|
| Un carácter a la izquierda | `←` |
| Un carácter a la derecha | `→` |
| Una palabra a la izquierda | `Ctrl` + `←` |
| Una palabra a la derecha | `Ctrl` + `→` |
| Un párrafo arriba | `Ctrl` + `↑` |
| Un párrafo abajo | `Ctrl` + `↓` |
| Una línea arriba | `↑` |
| Una línea abajo | `↓` |

## Tablas

| Acción | Atajo |
|----------|----------|
| Celda siguiente | `Tab` |
| Celda anterior | `Mayús` + `Tab` |

## Navegación rápida

| Acción | Atajo |
|----------|----------|
| Final de línea | `Fin` |
| Inicio de línea | `Inicio` |
| Pantalla siguiente | `Av Pág` |
| Pantalla anterior | `Re Pág` |
| Página siguiente | `Ctrl` + `Av Pág` |
| Página anterior | `Ctrl` + `Re Pág` |
| Final del documento | `Ctrl` + `Fin` |
| Inicio del documento | `Ctrl` + `Inicio` |

## Modo lectura

| Acción | Atajo |
|----------|----------|
| Pasar páginas | `Av Pág` / `Re Pág` |
| Desplazarse | Teclas de dirección o rueda del ratón |
