# 01-001: Course Introduction - Algorithms, Part I

## Course Overview

This course provides comprehensive coverage of algorithmic fundamentals through integrated learning resources and practical application.

### Materials

**Primary Resources**:

* **Lecture videos and slides**: Released weekly on course website
* **Programming assignments**: Hands-on development exercises
* **Job interview questions**: Real-world problem-solving scenarios
* **Algorithms, 4th edition textbook**: Primary reference material covering course content. [Here](https://www.amazon.com/gp/product/032157351X) and [HERE](../../../Algorithms_Part1/Algorithms_4thEd_.pdf)
* **Course booksite**: Open-access supplementary materials including textbook synopses and Java code utilised throughout the course

### Learning Approach

Success requires **active participation** rather than passive observation:

* **Write and debug code**: Practical programming experience
* **Solve problems**: Apply theoretical concepts to real scenarios
* **Study resources**: Engage with provided materials thoroughly
* **Participate in discussion forums**: Collaborate with peers and instructors

### Course Introduction

The **first programming assignment** introduces the Java programming model within the context of an important scientific application, establishing course spirit and pace expectations.

**Commitment**: Engage actively with lectures, assignments, resources, and community discussions to maximise learning outcomes.
